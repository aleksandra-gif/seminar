## Списки
Для того чтобы создать ненумерованные списки, нужно пункты выделить звездочкой. Например
* элемент 1
* элемент 2
* элемент 3

Для того чтобы добавить нумерованные списки, нужно их пронумеровать. Например:
1. первый пункт
2. второй пункт
